# AI Hub - AI效率提升工具集

> 汇聚AI驱动的效率工具、提示词与技能，助力开发效率飞跃

## 项目简介

AI Hub 是一个专注于利用AI技术提高工作效率的资源集合，提供经过精心设计的提示词（Prompts）、智能体（Agents）和工作流（Workflows），帮助用户在软件开发、架构设计、任务管理等场景中实现效率提升。

## 核心内容

### 1. 6A工作流

一套完整的多智能体协同方案，通过六个阶段的智能体团队协作，实现软件开发生命周期的规范化管理。

**六阶段流程**：

| 阶段 | 智能体 | 核心任务 |
|------|--------|----------|
| **Align** | AlgnAgent | 需求对齐，分析项目上下文、梳理需求边界、生成共识文档 |
| **Architect** | ArchAgent | 架构设计，分层设计、模块划分、接口定义 |
| **Atomize** | AtomAgent | 任务拆解，将架构转化为可执行的原子任务 |
| **Approve** | AprvAgent | 审批校验，校验需求、架构、任务的完整性和一致性 |
| **Automate** | AutmAgent | 编码执行，按任务依赖顺序实现代码 |
| **Assess** | AsseAgent | 评估验收，验证执行结果、评估项目质量 |

**核心特性**：
- 协调中心模式：统一入口、状态驱动、文档通信
- 灵活审批机制：支持自动/人工两种模式
- 平台无关性：可适配多种AI开发平台

### 2. 架构评审专家

独立的架构评审智能体，专注于项目代码库的深度逆向架构分析和全面检查。

**核心能力**：
- 项目全局扫描与技术栈识别
- 系统分层架构分析
- 模块依赖关系检测
- 架构坏味道识别
- 风险评估与健康评分

**适用场景**：
- 项目架构健康检查
- 代码质量评估
- 技术债务识别
- 重构规划支持
